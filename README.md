## 📅 프로젝트 개요
- **프로젝트 명:** AMR FLEET CORE
- **수행 기간:** 2026.04.20 ~ 2026.04.27
- **주요 기능**
  - **1:** TurtleBot 2대 기반 다중 물류 로봇 제어
  - **2:** A* 알고리즘 기반 경로 탐색 및 장애물 회피
  - **3:** 우선순위, 임계구역, 재탐색, 대피소 기반 충돌 예외 처리
  - **4:** Qt 관제 UI와 MariaDB를 통한 실시간 모니터링 및 작업 이력 관리
